Eukaryotic expression cloning

Eukaryotic cell(HEK,CHO,Baculovirus) cloning workbench

1. Input data: 아미노산 서열

FASTA 또는 일반 아미노산 서열을 입력하고, SignalP/TMHMM/ProP 분석 결과를 아래에 반영합니다.

대기
SignalP 6.0 signal peptide TMHMM 2.0 transmembrane domain ProP 1.0 R/K propeptide cleavage VectorBuilder codon optimization Thermo Fisher codon optimization

2. 발현 시스템 및 signal peptide/TM 선택

SignalP 결과에서 native signal peptide AA를 입력한 뒤 전략을 선택합니다. TMHMM 결과의 AA 범위는 제거할 수 있습니다.

아미노산 서열 적용 후 signal peptide/TM 처리 결과가 표시됩니다.
